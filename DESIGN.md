## Brand & Style

El sistema de diseño se fundamenta en la **confiabilidad técnica y la precisión digital**. Está diseñado para proyectar la imagen de un centro de servicio especializado en Santiago, donde la tecnología de vanguardia se encuentra con una atención al cliente impecable.

El estilo elegido es **Moderno-Corporativo con toques de Glassmorphism suave**. Se prioriza la legibilidad, la estructura lógica y una estética limpia que elimine la fricción visual para el usuario que busca soluciones técnicas urgentes. La interfaz debe sentirse robusta y estable, utilizando espacios en blanco generosos para denotar orden y profesionalismo, complementados con sutiles efectos de transparencia en capas superiores para evocar modernidad.

**Atributos de Marca:**
- **Resolutivo:** Orientado a la acción y la solución de problemas.
- **Transparente:** Información clara sobre servicios y procesos.
- **Especializado:** Estética que refleja conocimiento en hardware y software de alto nivel.

## Colors

La paleta cromática utiliza una jerarquía de alto contraste para guiar al usuario eficientemente.

- **Primario (Deep Navy):** Utilizado para componentes estructurales, encabezados y fondos de secciones de confianza. Representa la autoridad técnica.
- **Secundario (Blanco):** Actúa como lienzo principal, garantizando que el contenido respire y la interfaz se sienta "limpia" (asociado a laboratorios técnicos).
- **Acento Técnico (Tech Cyan):** Reservado para elementos interactivos secundarios, enlaces informativos e iconos de soporte.
- **Acción Crítica (WhatsApp Green):** Exclusivamente para botones de contacto directo, cotizaciones por chat y estados de "En línea" o "Listo".
- **Neutros:** Se utiliza un gris muy claro (`#F7F8F9`) para separar contenedores y un gris casi negro (`#222222`) para garantizar la máxima legibilidad tipográfica.

## Typography

Se ha seleccionado una combinación de tipografías que equilibra el carácter tecnológico con la legibilidad funcional.

- **Titulares (Hanken Grotesk):** Una sans-serif geométrica contemporánea. Su estructura nítida y moderna comunica precisión técnica. Se aplica con pesos fuertes (600-700) para establecer una jerarquía clara.
- **Cuerpo de texto (Inter):** La referencia en interfaces modernas por su excelente legibilidad en pantallas de cualquier tamaño. Se utiliza para descripciones de servicios, artículos de blog y formularios.
- **Etiquetas y Micro-copy:** Se utiliza Inter con tracking (espaciado entre letras) ligeramente aumentado y peso semi-bold para botones y categorías, facilitando el escaneo visual rápido.

## Layout & Spacing

El sistema utiliza una **retícula fluida de 12 columnas** para escritorio y una retícula de 4 columnas para dispositivos móviles.

- **Riesgo de Espaciado:** Se basa en una unidad de 8px (base-8 system). Todos los paddings y márgenes deben ser múltiplos de 8.
- **Contenedores:** El contenido principal se aloja en un contenedor centrado de máximo 1200px para evitar líneas de texto excesivamente largas en monitores ultra-wide.
- **Ritmo Vertical:** Se prefiere el uso de "Stacking" (espaciado consistente entre elementos de una misma categoría) con 32px de separación entre secciones principales para mantener la claridad.
- **Adaptabilidad:** En dispositivos móviles, los márgenes laterales se reducen a 16px para maximizar el área de lectura, y los elementos de tarjetas suelen ocupar el ancho total de la pantalla.

## Elevation & Depth

Para comunicar modernidad sin sacrificar la sobriedad, el sistema emplea **capas tonales y sombras ambientales difusas**:

- **Nivel 0 (Fondo):** Superficie `#FFFFFF` o `#F7F8F9`.
- **Nivel 1 (Tarjetas/Contenedores):** Fondo blanco con una sombra muy suave (Blur: 20px, Spread: 0, Color: `rgba(10, 25, 47, 0.05)`). Esto crea una sensación de elevación natural.
- **Nivel 2 (Modales/Pop-overs):** Elevación superior con bordes finos de 1px en color `#0A192F` con opacidad al 10% para definir límites sin saturar.
- **Interacción:** Al pasar el cursor (hover) sobre elementos interactivos, la sombra debe expandirse ligeramente y el elemento puede elevarse -2px en el eje Y para dar feedback de "clicabilidad".

## Shapes

El lenguaje de formas es **redondeado pero controlado**, evitando círculos excesivos para mantener la seriedad profesional.

- **Componentes Estándar:** Los botones, campos de entrada y tarjetas utilizan un radio de 0.5rem (8px).
- **Contenedores Grandes:** Secciones destacadas o modales pueden escalar hasta 1rem (16px) para suavizar la estética general.
- **Iconografía:** Se deben utilizar iconos de línea fina (2px de grosor de trazo) con terminaciones redondeadas para coherencia con la tipografía.

## Components

- **Botones:** 
  - *Primario:* Fondo `#0A192F`, texto blanco. Para acciones de navegación principal.
  - *Acción (WhatsApp):* Fondo `#25D366`, texto blanco, icono de marca a la izquierda. Es el botón con más peso visual.
  - *Tecnológico:* Fondo `#00B4D8`, ideal para selectores de servicio o cotizadores.
- **Tarjetas (Cards):** Fondo blanco, bordes redondeados (8px), sombra suave nivel 1. Se utilizan para listar servicios (ej. "Cambio de Pantalla", "Reparación de Placa"). Incluyen un icono lineal de color `#00B4D8` en la parte superior izquierda.
- **Inputs:** Bordes de 1px color gris claro, que cambian a `#00B4D8` al ganar el foco. Etiquetas siempre visibles en `label-bold`.
- **Chips de Estado:** Pequeñas etiquetas redondeadas para indicar disponibilidad o tipo de dispositivo (Mac, PC, Notebook). Fondo semi-transparente del color de la categoría.
- **Listas Técnicas:** Uso de checkmarks en `#25D366` para enumerar beneficios o puntos incluidos en un servicio técnico, reforzando la positividad y confianza.
- **Barra de Navegación:** Fondo blanco con un "Backdrop Blur" si se desplaza sobre el contenido, manteniendo el logo a la izquierda y el CTA de contacto destacado a la derecha.
